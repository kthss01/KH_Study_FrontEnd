폼관련 태그

form 태그는 html 사용자가 입력할 수 있는 양식을 제공하는 태그이다. form 태그내의 input 태그들을 통해 사용자가 입력한 정보를 서버로 넘기면서 요청하는 역할을 한다. * action 속성 : 폼내에 입력된 값들을 전달할 서버측 URL을 기입한다.
* method 속성 : get/post 방식으로 전송 방식을 지정한다.
> get : 서버에 사용자가 입력한 값을 전달하면서 요청시 사용자가 입력한 값들이 url에 노출되는 방식 > post : 서버에 사용자가 입력한 값을 전달하면서 요청은 하되 url에 노출되지 않음

 

submit 버튼을 눌렀을 때 method 요청 전송방식으로 action에 지정한 경로로 사용자가 입력한 값을 (key : value) 형태로 전달한다.
